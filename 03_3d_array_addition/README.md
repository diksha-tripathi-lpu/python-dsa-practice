# 3D Array Addition

## Problem Statement

Write a Python function that takes two three-dimensional numeric data sets and adds them componentwise.

## Description

This program performs component-wise addition of two three-dimensional arrays. It first checks whether both arrays have the same dimensions. If they do, it adds the corresponding elements and returns the resulting 3D array.

## Algorithm

1. Check if both arrays have the same dimensions.
2. Traverse each layer, row, and column using nested loops.
3. Add corresponding elements from both arrays.
4. Store the sum in a new 3D array.
5. Display the resultant array.

## Sample Output

```
Resultant 3D Array:
[[11, 22], [33, 44]]
[[55, 66], [77, 88]]
```
